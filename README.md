# Swiggy-Instamart-Sales-Dashboard
Swiggy Instamart Sales Dashboard (Power BI)  An interactive Power BI dashboard designed to analyze Swiggy Instamart sales performance, orders, ratings, shop categories, and city tiers. The dashboard provides actionable insights through dynamic filters, KPI tracking, trend analysis, and visual reports to support data-driven business decisions. 🚀
